* Spring+SpringMVC+MyBatis\Redis 服务端
* Administrator 网页端
* IOS+ANDROID 客户端

## <a id="What's_features:"></a>What's features:
* Swagger2
* AdminLTE
* Amis
* ArcSoft
* RongCloud
* QiniuCloud
* JWT
* Shiro
* WebSocket
* WebService
* TxManager
* Mybatis-plus
* UnitTest
